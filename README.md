# 🐴 Polo Horses Management System

A modern, responsive web application for managing polo horse data with a beautiful interface. Collect detailed information about polo horses and generate professional summaries and analytics.

## Features

✨ **Key Features:**
- 📝 Intuitive form to add polo horses with detailed information
- 📊 Real-time statistics and fleet analytics
- 📋 Detailed horse registry table with sorting capability
- 📄 AI-generated prose summary of your polo horse fleet
- 🎨 Beautiful, responsive design with equestrian aesthetic
- 💾 Local data persistence (uses browser storage)
- 📱 Mobile-friendly interface

## Data Collected

- **Horse Name** - Unique identifier for each horse
- **Age** - Horse age in years
- **Breed** - Selection from popular polo horse breeds
- **Weight** - Horse weight in kilograms
- **Owner** - Owner's name
- **Polo Games Played** - Number of competitive games
- **Last General Vet Check** - Date of most recent veterinary examination

## Breeds Supported

- Argentinian Criollo
- Thoroughbred
- Quarter Horse
- Arab
- Cross

## Getting Started

### Quick Start (Browser Only)

1. Clone this repository
2. Open `index.html` in your web browser
3. Start adding horses!

### Installation

```bash
git clone https://github.com/Pomare3/polo-horses-app.git
cd polo-horses-app
# No build process needed - just open index.html
```

## Usage

### Adding a Horse

1. Fill in the form fields with horse information
2. All fields marked with * are required
3. Click "Add Horse" to save
4. The horse will appear in the registry table

### Viewing Statistics

- Fleet statistics display automatically when horses are added
- See total count, averages, and aggregate data
- Statistics update in real-time

### Viewing Summary

- Professional prose summary generates automatically
- Includes fleet overview, breed diversity, and recent vet check information

### Managing Records

- View all horses in the detailed registry table
- Delete a horse record using the ✕ button
- Hover over rows for better visibility
